New Fljud Released

by Binaryape - Sep 15th, 2002 - posted in Ancient Archives

I’ve released a new version of Fljud onto Sourceforge. This one’s mainly bug fixes and a bit more documentation. I’ve been careful to check that lots of different browsable folder configurations are possible, but I forgot to check that users could configure only one, or even none at all. Doh. I’m leaving Fljud alone for a while now so I can work on Fljauth and Smartcast instead.
